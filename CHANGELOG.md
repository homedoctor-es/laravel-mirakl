# Changelog

All notable changes to `laravel-mirakl` will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release
- Laravel service provider for Mirakl SDK integration
- Facade for convenient access
- Configuration file with environment variable support
- Helper class with common operations
- Porto Pattern examples (Tasks, Actions, Commands, Models)
- EventBridge integration examples
- Comprehensive documentation
- Automatic retry on rate limits
- Pagination helper methods

### Features
- Support for PHP 8.1, 8.2, and 8.3
- Support for Laravel 10.x and 11.x
- Auto-discovery of service provider
- Complete wrapper around Mirakl Shop API Client
- Examples for offers, products, and orders synchronization
